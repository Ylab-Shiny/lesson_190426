3.1 変数とデータ型

  • 数値

  • 文字列

  • ベクトル
  • 行列
  • リスト

  • データフレーム

  • NULL
  • TRUE(T) / False(F)
  • NA(欠損値)
  • NaN(非数)

  • Inf(無限大)

# 代入演算子(<-)や演算子(+, -, *, /)の両脇に半角スペースを入れるとよい（コードが見やすくなる）
x <- 5
x
## [1] 5
# データ型を確認するときはclass関数を使う numeric:数値型 character:文字列型
class(x)
## [1] "numeric"
x + 3
## [1] 8
# まず先に定義した変数mdatのみを実行環境(Environment)から削除
rm(list = "mdat")
# すると、実行環境にはまだ変数xが残る
ls()
## [1] "x"
# 全ての変数を削除したい場合は…
rm(list = ls())
ls()
## character(0)
